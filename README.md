# ALter

Site for various powerful AniList tools, focusing on bulk editing.  
The name stands for **A**ni**L**ist **T**ools for **E**diting **R**apidly.  

## Features

- **Scorer**  
  Bulk edit scores with keyboard shortcuts.

- **Dropper**  
  Bulk drop entries not recently updated.

### Planned

- **Fixer**  
  Fix incorrect data like start/end date and progress.

- **Noter**  
  Bulk edit notes with find/replace/etc.

- **Relater**  
  Find related entries and bulk add to planning.
